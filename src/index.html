Привет!

Меня зовут Евгений Бояркин, и это мой дипломный проект в рамках обучения на образовательной платформе Яндекс.Практикум.

Здесь я применяю навыки HTML и CSS верстки, использую JavaScript для управления элементами, обращаюсь к API и получаю данных с сервера. Проект собран при помощи библиотек Webpack и размещен на GitHub.

В течение семи месяцев я осваивал профессию frontend-разработчика. Разбирался в кроссбраузерной семантической верстке, адаптивном дизайне, объектно-ориентированном программировании. Учился понимать язык браузеров и делать код красивым. И для меня важна не только внешняя красота, но и внутренняя эстетика!
